# App Support

Support information for apps by Kiran Kashapwar.

## Tic Tac Toe - Retro

Tic Tac Toe - Retro is a local board game app for iOS. It supports quick matches, local two-player games, computer opponents, different board sizes, sound, music, and ads served through Google AdMob.

### Getting Help

If you need help with Tic Tac Toe - Retro, please open a public support issue in this repository:

https://github.com/k1rank94/support/issues

When reporting a problem, include:

- App name: Tic Tac Toe - Retro
- iPhone or iPad model
- iOS version
- App version
- A short description of what happened
- Steps to reproduce the issue, if possible

Please do not include passwords, payment details, private addresses, or other sensitive personal information in GitHub issues.

### Common Questions

#### The game sound or music is not playing

Check the in-app sound controls and make sure your device volume is enabled. Also check whether Silent Mode, Focus, or Bluetooth audio routing is affecting playback.

#### Ads are not loading

Ads may not appear immediately in every region or network condition. If ads do not load, you can continue playing the game normally.

#### I want to change privacy choices

Where privacy options are required, the app provides a privacy options control so you can review or change your advertising choices.
